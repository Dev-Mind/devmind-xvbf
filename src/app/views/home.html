Tests et browser headless

Le but de ce projet est de montrer comment se passer de PhantomsJS lorsque vous voulez construire votre projet sur des serveurs ne contenant aucun serveur X.

Si vous utilsiez vos propres serveurs d'intégration continues, les documentations nous poussent souvent à utiliser le browser headless PhantomJS qui a la particularité de nous fournir un moteur Javascript sans interface graphique. C'est très bien pour les tests unitaires mais le problème c'est que ce n'est pas un vrai navigateur. Comment utiliser le moteur Javascript de Chrome ou de Firefox ?

J'essaye ici d'illustrer la résolution de ce problème via Xvfb
